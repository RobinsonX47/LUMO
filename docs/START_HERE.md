## 2) Production Setup

- TESTING_GUIDE.md for OAuth and end-to-end testing scenarios.

## 5) Current Capability Snapshot

- Authentication: email/password + Google OAuth
- Content: movies, TV, anime, genres, detailed pages
- User: profile editing, avatars, watchlist, reviews, continue watching
- Social: public profiles, follow/unfollow, directory, search, notifications
- Ops: health and readiness endpoints, rate limiting, security headers
